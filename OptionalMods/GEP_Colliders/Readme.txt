## GEP_Colliders

GEP_Colliders is an optional mod that add colliders to terrain scatters such as boulders and trees. WARNING! Enabling colliders can have an adverse affect on game performance.

To install GEP_Colliders:

1. Drill down to [GEP Download]\OptionalMods\GEP_Colliders\ .

2. Copy/merge the GameData folder into your KSP install.
